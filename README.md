#  Optimisation de la Flotte de Taxis de NYC

###  Objectif
Analyser les données massives de NYC (2021-2024) pour optimiser l'allocation des véhicules et réduire l'impact environnemental.

###  Méthodologie
1. **Acquisition & Nettoyage** : Traitement des données brutes avec Pandas.
2. **EDA (Analyse Exploratoire)** : Identification des zones de forte demande.
3. **Visualisation** : Création de cartes de chaleur avec Matplotlib/Seaborn.

### Impact & Résultats
Recommandations stratégiques permettant une réduction théorique de 15% des trajets à vide.
